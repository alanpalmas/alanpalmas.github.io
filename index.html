Sobre mí

Mi primera experiencia como QA Tester fue en el bootcamp TripleTen, donde tuve la oportunidad de trabajar en equipo con profesionales experimentados en aseguramiento de calidad de software. A través de esta experiencia, mis habilidades técnicas, analíticas, de pensamiento crítico y trabajo en equipo se vieron potenciadas enormemente. Después, decidí profundizar en el área de aseguramiento de calidad y optimización de procesos de software, por la cual siento una gran pasión.

Con el tiempo, mi enfoque se ha centrado en aprender y mejorar en automatización, aplicando las mejores prácticas y técnicas de testing. Mis sólidos conocimientos en pruebas manuales y automatizadas, así como mi experiencia en el desarrollo backend adquiridas en el programa Oracle Next Education, me permiten brindar soluciones eficientes y creativas.

Fuera de mi rutina laboral, disfruto escuchando música, jugando videojuegos, leyendo sobre nuevas tecnologías o pasando el tiempo con mis mascotas.

Proyectos

Proyecto #1 - Urban Routes

Web Testing

Este proyecto de QA para el sitio web de Urban Routes involucró: validación de diseños contra especificaciones de Figma, creación de casos de prueba, ejecución de pruebas manuales de UI, usuario final y nuevas funcionalidades. Identifiqué y reporté más de 50 bugs en Jira. Apliqué técnicas avanzadas con Charles Proxy para insertar la nueva funcionalidad mediante breakpoints.

JavaScript, Devtools, JIRA, Charles Proxy

Proyecto #2 - Urban Grocers

API Testing

Pruebas de API para gestión de kits, entregas y carrito en Urban Grocers, aseguré la calidad y funcionalidad óptima de estas características clave. Estudié la documentación de API para planificar y ejecutar pruebas de API, diseñando casos de prueba exhaustivos. Utilicé Postman para pruebas de API y reporté bugs en Jira.

Postman, JIRA, JSON, ApiDoc

Proyecto #3 - Urban Lunch

Mobile Testing

Este proyecto de QA para la aplicación móvil Urban Lunch involucró: pruebas exhaustivas en la plataforma Android, con la responsabilidad principal de garantizar la calidad y funcionalidad óptima. Realicé pruebas utilizando Android Studio y dispositivos reales, aplicando buenas prácticas de testing sobre aplicaciones móviles nativas. Identifiqué y reporté bugs y defectos en Jira.

Android Studio, JIRA

Proyecto #4 - Urban Grocers

API Testing Automated

Pruebas automatizadas con Python, pytest y requests para validar la creación de usuarios y kits en la aplicación web, asegurando la calidad y funcionalidad óptima. Automaticé pruebas diseñadas por el equipo de QA. Identifiqué y resolví desafíos técnicos, aplicando buenas prácticas de QA para garantizar la integridad de las pruebas.

Python, Pytest, Requests

Proyecto #5 - Urban Routes

Web Testing Automated

Desarrollé pruebas automatizadas, con la responsabilidad principal de asegurar el correcto funcionamiento de la configuración de rutas, selección de tarifas, verificación de números de teléfono y gestión de métodos de pago. Implementé una arquitectura de Página de Objeto (POM) para mantener un código modular y mantenible.

Python, Pytest, Selenium WebDriver

Proyecto #6

Decodificador de texto

Desarrollé una aplicación web interactiva utilizando JavaScript, HTML y CSS que permite cifrar y descifrar mensajes de texto de acuerdo con reglas de reemplazo de letras establecidas. Implementé una lógica eficiente con funciones y algoritmos en JavaScript.

HTML, CSS, JavaScript
